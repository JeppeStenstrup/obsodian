# Model-View-Controller Pattern

## [[JHotDraw Outline|JHotDraw]]
Model View Controller Pattern
- Model
	- Figures (and their attributes: FillColor, Position)
	- Drawing (figure container)
- View
	- DrawingView (clipping view of a window)
	- DrawingWindow
- Controller
	- Tools to manipulate the model
