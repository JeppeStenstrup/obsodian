# NLP Intro

## What is NLP?
A subfield of linguistics, computer science, and [[Artificial Intelligence|AI]].

The study of giving computers the ability to understand written and spoken language.

## Main fields of study
- Speech Recognition
- Speech-to-text
- Translation
- Text generation
- **[[Text Classification]]** -- The focus of this subject
	- Word representation
		- Word embeddings
	- Sentence representation
		- [[CNN]], [[RNN]]

## Use cases of NLP
- Search engines
- Span detection
- Chat bots
- Smart speakers
- Social media text classification
	- Hate speech
- Github Copilot, OpenAI Codex
- Medical domain
	- Text summarization of EHR
	- Text extraction from EHR
